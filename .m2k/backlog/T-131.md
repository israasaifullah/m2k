# T-131: Display output rate indicator (KB/s)

**Epic:** EPIC-024

## Description
Add real-time output rate indicator to terminal UI. Shows current KB/s throughput and visual indicator when throttling is active (heavy load).

## Acceptance Criteria

- Show ğŸ¢ icon when throttle > 33ms (heavy load detected)

## Technical Notes
**File:** `src/components/Terminal.tsx`

**Approach:**
```typescript
const [outputRate, setOutputRate] = useState(0);
const [isThrottled, setIsThrottled] = useState(false);

// In output listener:
let bytesThisSecond = 0;
setInterval(() => {
  setOutputRate(bytesThisSecond / 1024); // KB/s
  setIsThrottled(bytesThisSecond > 50_000); // >50KB/s
  bytesThisSecond = 0;
}, 1000);
```
