# T-133: Implement throttle adjustment logging

**Epic:** EPIC-024

## Description
Add debug logging for adaptive throttle adjustments. Logs when emit_delay changes, bytes/sec, and throttle decisions. Enable via environment variable or debug flag.

## Acceptance Criteria
- Log throttle changes: `[PTY] Throttle adjusted: 16ms -> 33ms (bytes/s: 75000)`
- Log emit events: `[PTY] Emitted batch: 8192 bytes (delay: 33ms)`
- Controlled by `DEBUG_TERMINAL=true` env var
- No performance impact when disabled
- Use Tauri logging system (`log::debug!`)

## Technical Notes
**File:** `src-tauri/src/pty.rs`

**Approach:**
```rust
let mut last_delay = Duration::from_millis(16);

// When emit_delay changes:
if emit_delay != last_delay {
    log::debug!(
        "[PTY-{}] Throttle adjusted: {}ms -> {}ms (bytes/s: {})",
        pty_id, last_delay.as_millis(), emit_delay.as_millis(), bytes_per_sec
    );
    last_delay = emit_delay;
}

// On emit:
log::debug!(
    "[PTY-{}] Emitted batch: {} bytes (delay: {}ms)",
    pty_id, batch.len(), emit_delay.as_millis()
);
```

**Enable:** `RUST_LOG=debug cargo tauri dev`
