# T-026: Animations and Transitions

**Epic:** EPIC-007

## Description
- Add smooth transitions for card movements
- Animate status changes (backlog → in-progress → done)
- Add hover states and micro-interactions
- Ensure animations enhance UX without being distracting

## Acceptance Criteria
- Smooth transitions when moving cards between columns
- Subtle hover effects on interactive elements
- Loading states animated appropriately
- Performance maintained (no jank)

## Technical Notes

- Follow Geist animation guidelines
- Keep animations under 300ms for snappiness

## Testing

- Check performance with many cards
- Ensure animations respect prefers-reduced-motion
