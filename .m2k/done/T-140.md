# T-140: Replace copy relative path into copy full path

**Epic:** EPIC-031

## Description
- In m2k-app, we have multiple features that allow user to copy the relative path.
- The features that have this function includes (but not limited to):
  - Copy resources path
  - Copy epic path
  - Copy ticket path
  - Insert resources during PRD Mode
- Replace all these copy relative path function into copy full path of the file
- The purpose of this implementation is to allow LLM to have the exact path to the reference file without 
doing unnecessary search

## Acceptance Criteria
- Whenever a trigger button that copy path is clicked, it should copy the fullpath instead of the relative path
