# T-132: Add clear buffer action for runaway processes

**Epic:** EPIC-024

## Description
Add "Clear Output" button to terminal header. Allows users to manually clear xterm buffer during runaway processes without killing the PTY.

## Acceptance Criteria
- Add "Clear" button to terminal header
- Clicking clears xterm display (`xterm.clear()`)
- Does NOT kill the running PTY process
- Clears frontend buffers (writeBuffer, pausedBuffer)
- Keyboard shortcut: Ctrl+L (standard terminal clear)

## Technical Notes
**File:** `src/components/Terminal.tsx`

**Approach:**
```typescript
const handleClear = () => {
  xterm.clear();
  writeBuffer = '';
  pausedBuffer = '';
};

// Add keyboard listener
xterm.onKey((e) => {
  if (e.domEvent.ctrlKey && e.domEvent.key === 'l') {
    handleClear();
  }
});
```

**UI Position:** Terminal header, between pause and close buttons

## Dependencies
- None

## Testing
- Run heavy output command
- Click "Clear" button
- Verify display clears but process continues
- Test Ctrl+L shortcut
- Confirm buffers are cleared
