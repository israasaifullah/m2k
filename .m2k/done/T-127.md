# T-127: Implement sliding window tracker in pty.rs

**Epic:** EPIC-024

## Description
Add sliding window data structure to track PTY output bytes over time. Use VecDeque to store (timestamp, bytes_read) tuples and retain only entries from the last 1 second for throughput calculation.

## Acceptance Criteria
- Add `VecDeque<(Instant, usize)>` to track recent bytes
- Implement auto-pruning: retain only entries where `elapsed() < 1s`
- Calculate total bytes/sec over the sliding window
- No memory leak (window size capped)

## Technical Notes
**File:** `src-tauri/src/pty.rs` (around line 77-82)

**Approach:**
```rust
use std::collections::VecDeque;

let mut recent_bytes: VecDeque<(Instant, usize)> = VecDeque::new();

// In read loop:
recent_bytes.push_back((Instant::now(), n));
recent_bytes.retain(|(t, _)| t.elapsed() < Duration::from_secs(1));

let bytes_per_sec: usize = recent_bytes.iter().map(|(_, b)| b).sum();
```
