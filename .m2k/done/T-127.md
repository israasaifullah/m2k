# T-127: Implement project name detection

**Epic:** EPIC-023

## Description
Detect and extract unique project name for creating project-specific backup folders.

## Acceptance Criteria
- Extract project name from project path
- Handle duplicate project names across different paths
- Generate unique identifier when needed
- Project name suitable for folder naming (no invalid chars)

## Technical Notes
- Use last directory name as project name
- Sanitize project name for filesystem compatibility
- Consider adding hash/UUID suffix if name collision detected
- Example: `/path/to/my-project` → `my-project`

## Dependencies
- None

## Testing
- Various project paths → correct names extracted
- Special characters → properly sanitized
- Duplicate names → unique identifiers added
