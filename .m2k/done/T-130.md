# T-130: Add backup status indicators

**Epic:** EPIC-023

## Description
Show visual feedback for backup operations and status.

## Acceptance Criteria
- Display backup in progress indicator during sync
- Show success notification when backup completes
- Show error notification when backup fails
- Indicate if backup path is configured
- Display last sync timestamp (optional)

## Technical Notes
- Use toast/notification system for feedback
- Loading spinner during sync operation
- Visual indicator in UI showing backup path status
- Consider adding backup icon/badge to project list

## Dependencies
- T-128

## Testing
- Start sync → loading indicator appears
- Sync completes → success message shown
- Sync fails → error message with details shown
- No backup path set → appropriate indicator shown
