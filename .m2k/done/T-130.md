# T-130: Add pause/resume button to Terminal UI

**Epic:** EPIC-024

## Description
Add pause/resume controls to Terminal component header. When paused, buffer output in memory without rendering. Resume flushes buffered output at controlled rate.

## Acceptance Criteria
- Add pause/resume toggle button to terminal header
- Pause: stop calling `xterm.write()`, accumulate in buffer
- Resume: flush buffer with controlled playback
- Visual state: show "Paused" indicator when active
- Keyboard shortcut: Ctrl+P to toggle

## Technical Notes
**File:** `src/components/Terminal.tsx`

**Approach:**
```typescript
const [isPaused, setIsPaused] = useState(false);
let pausedBuffer = '';

const flushBuffer = () => {
  if (isPaused) {
    pausedBuffer += writeBuffer;
    writeBuffer = '';
    return;
  }

  if (writeBuffer) {
    xterm.write(writeBuffer);
    writeBuffer = '';
  }
};

const onResume = () => {
  setIsPaused(false);
  xterm.write(pausedBuffer);
  pausedBuffer = '';
};
```

**UI Position:** Terminal header, right side near close button
